# Brief de diseno UI/UX

## Personalidad visual

La app debe sentirse moderna, clara y confiable. La interfaz debe priorizar lectura rapida de datos, comparacion visual y navegacion sencilla.

## Principios de diseno

- Claridad antes que decoracion.
- Datos importantes visibles en el primer viewport.
- Colores climaticos utiles, no solo esteticos.
- Componentes responsivos para movil y desktop.
- Estados vacios y errores explicados con lenguaje simple.

## Paleta inicial

- Azul profundo: encabezados, navegacion y fondo principal.
- Cian: informacion de temperatura y clima activo.
- Verde: precision alta o coincidencia entre fuentes.
- Amarillo: advertencias o diferencias moderadas.
- Rojo suave: alertas o diferencias grandes.
- Gris claro: superficies, bordes y texto secundario.

## Tipografia

- Inter, system-ui o similar.
- Titulos claros y compactos.
- Numeros de temperatura con peso visual alto.
- Texto tecnico en tamanos moderados para no saturar.

## Componentes clave

- Tarjeta de clima actual.
- Barra de busqueda.
- Selector de unidades Celsius/Fahrenheit.
- Mapa interactivo.
- Tabs para actual, horario, semanal y comparacion.
- Chips de estado para precision alta, media o baja.
- Graficas de tendencia.

## Experiencia movil

- Priorizar clima actual y busqueda.
- Mapa accesible en una pantalla separada o seccion expandible.
- Botones grandes y legibles.
- Evitar tablas anchas; usar tarjetas compactas.
